# Documentation Complementaire

Ce dossier contient les documents de support du projet :
- notes d'hypotheses ;
- decisions de modelisation ;
- comptes-rendus d'analyse ;
- references metier et techniques.

## Bonnes pratiques
- Nommer les documents avec date et sujet (`YYYY-MM-DD_sujet.md`).
- Documenter chaque changement de methode ou hypothese.
- Referencer les fichiers de donnees et scripts utilises.
